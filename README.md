# Agent-model
Cистема моделирования поведения группы БПЛА, интегрирующей метод роя частиц и математический аппарат путей Дабинса для решения задач кооперативного планирования траекторий

## Системные требования
- MATLAB R2025a или новее
- 8 ГБ ОЗУ (минимум 4 ГБ)
- Windows 10/11, Linux или macOS

## Установка и запуск
1. Клонируйте репозиторий в любую папку на жёстком диске (например, C:\Users\%USERNAME%\Documents\)
2. Откройте MATLAB R2025a
3. Перейдите в папку:
- В MATLAB нажмите кнопку "Обзор папки" или введите в командной строке:

  ```cd 'C:\путь\к\Agent_model'```
- Или дважды щёлкните на папке Agent_model в окне "Текущая папка" MATLAB
4. Запустите главный скрипт Demo1.m одним из способов:
- В командной строке MATLAB (нижнее окно с >>) введите:

  ```Demo1```

  и нажмите Enter
- Или откройте файл Demo1.m двойным щелчком в окне "Текущая папка", затем нажмите кнопку Run (зелёный треугольник) на панели редактора или клавишу F5

Примечания:
- Перед запуском убедитесь, что рабочей папкой MATLAB является Agent_model
- Все вспомогательные файлы должны оставаться в исходной структуре папок
